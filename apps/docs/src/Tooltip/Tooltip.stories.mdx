## 설치

프로젝트에 @custompackages/designsystem이 설치되어 있는지 확인하세요. 

다음과 같이 Tooltip 컴포넌트를 임포트할 수 있습니다

```jsx
import { Tooltip } from '@custompackages/designsystem';
```

      </Tooltip.Trigger>

      </Tooltip.Content>
    </Tooltip>
  )
}
